# Copilot Calculator

Copilot Calculator is a simple web application for performing basic arithmetic operations: addition, subtraction, multiplication, and division. Built with Python and Streamlit, it provides a fast and interactive user experience.

## Features
- Add, subtract, multiply, or divide two numbers (supports decimals)
- User-friendly web interface powered by Streamlit
- Error handling for invalid operations (e.g., division by zero)
- Instant results without loading screens

## How It Works
- Enter two numbers in the input boxes
- Select the desired operation from the sidebar
- Click the "Calculate" button to see the result
- If division by zero is attempted, a warning or error message is displayed

## Technology
- **Frontend:** Streamlit for UI and user interaction
- **Backend:** Python functions for calculation logic

## Getting Started
1. Create and activate a Python virtual environment
2. Install required libraries: `pip install -r requirement`
3. Run the app: `streamlit run app.py`
